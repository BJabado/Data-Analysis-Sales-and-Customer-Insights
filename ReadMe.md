# Data Analysis: Sales & Customer Insights

## About the Project
This project was a great way for me to get hands-on with data analysis using Python. I worked with three separate CSV files—customers, orders, and products—to simulate a real-world relational database scenario. My goal was to link these datasets together to uncover key business insights, like who our top customers are and which product categories are bringing in the most revenue.

## What I Learned (Technical Skills)
- **Data Wrangling:** I learned how to handle messy file paths and merge relational data effectively using Pandas, mimicking SQL JOIN operations.
- **Python for Analysis:** I moved beyond basic scripts to calculate metrics and group data to extract trends.
- **Visualization:** I used Matplotlib to transform raw numbers into a clear bar chart, which makes the findings much easier to present.
- **Troubleshooting:** I spent time debugging data ingestion issues, which taught me how to handle naming inconsistencies and path management in a real environment.

## How to Run
1. Make sure you have Python installed.
2. Install the required libraries: 
   `pip install pandas matplotlib`
3. Run the analysis script: 
   `python analysis.py`

## Key Takeaways
After processing the data, I identified the top-spending customers and visualized the revenue trends. The analysis clearly shows that the **Home** and **Sports** categories are currently the biggest revenue drivers.
