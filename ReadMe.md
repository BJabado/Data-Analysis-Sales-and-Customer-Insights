**Sales & Customer Insights**
This project demonstrates relational data analysis using DuckDB and Python. It transforms raw CSV data into business insights by performing SQL-based joins and aggregations.

**Skills**
-SQL Engineering: Joins and aggregations directly on CSV datasets via DuckDB.

-Data Engineering: Robust file path management for cross-platform compatibility.

-Data Visualization: Revenue trends visualized with Matplotlib.

**Quick Start**
Install dependencies:

pip install duckdb matplotlib
Run the analysis:

python analysis.py

**Key Insights**

-Customer Value: Automated identification of top spenders.

-Category Performance: Real-time revenue reporting per product category.
